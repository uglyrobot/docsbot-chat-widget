Server: sign JWT with Stripe customer ID (Node)

Use your Widget embed signature key as the HMAC secret. This page also mints a test JWT in the browser (see DOCSBOT_TEST_SIGNATURE_KEY in the script below)—for production, sign only on your server. 

const jwt = require('jsonwebtoken')

const teamId = 'nG4F5A3BFSBzdYc5TZIX'
const botId = 'uy8srweloFNgRadNtwvf'
const stripeCustomerId = 'cus_MjCntOVjGjAqYq' // never put this in client identify/metadata

const payload = {
  exp: Math.floor(Date.now() / 1000) + 3600,
  iat: Math.floor(Date.now() / 1000),
  team_id: teamId,
  bot_id: botId,
  metadata: {
    priv_stripe_customer_id: stripeCustomerId,
  },
}

const token = jwt.sign(payload, process.env.DOCSBOT_SIGNATURE_KEY, { algorithm: 'HS256' })
// Pass token to the page as init.signature
