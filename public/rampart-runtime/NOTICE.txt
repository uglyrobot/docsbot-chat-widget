Rampart runtime artifacts
=========================

License: Creative Commons Attribution 4.0 International (CC BY 4.0)

DocsBot self-hosts this runtime from the widget CDN for optional client-side PII
redaction. The hosted runtime copy exposes a small configureTransformersEnv
helper so the widget can force Transformers.js to load the self-hosted model
files and avoid remote Hugging Face requests.
