# Public Accessibility Statement Draft

DocsBot is committed to improving accessibility for users of the DocsBot Chat Widget. The widget is being evaluated against WCAG 2.1 Level AA criteria, and we are continuing to remediate issues as they are identified.

In the current evaluated widget scope, DocsBot supports keyboard interaction for core chat flows and includes accessibility improvements for focus visibility, control labeling, and form validation. Some areas still require further testing, including direct assistive-technology validation and additional review of embedded third-party experiences.

If you encounter an accessibility barrier while using the DocsBot Chat Widget, please contact our support team so we can investigate and prioritize a fix. Accessibility work is ongoing, and we expect to keep refining both the product and our testing process over time.
